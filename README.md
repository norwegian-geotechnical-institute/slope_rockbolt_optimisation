# slope_rockbolt_optimisation
Integration of Rhino/Grasshopper and Rocscience RocSlope for evaluation of rock block stabilities and bolting optimisation

## Software
### Rhino
- Rhino Version 7 SR34 (7.34.23267.11001, 2023-09-24)
